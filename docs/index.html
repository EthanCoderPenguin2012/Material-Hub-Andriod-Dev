🚀 Getting Started

  1. Clone the repository: 
    git clone https://github.com/EthanCoderPenguin2012/Material-Hub-Android-Dev.git
  2. Open the project in Android Studio.
  3. Build and run the app on your emulator or physical device.
  4. Explore the features and provide feedback!
Prerequisites: Android Studio Meerkat+, JDK 21+, Android 8.0+, Git

🛠️ Technologies Used

Kotlin Android Jetpack Material Design 3 Retrofit Room
